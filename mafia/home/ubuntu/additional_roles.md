# 추가 구현할 역할 목록

## 시민팀 역할
1. **버스기사 (BusDriver)**
   - 밤에 두 사람을 지목해 받는 결과를 바꿈
   - 우선순위: 의사(60)보다 높게 설정 (예: 70)
   - 구현 방법: 두 플레이어의 밤 행동 결과를 서로 바꿈

2. **정신병자 (Psycho)**
   - 시민팀 직업의 능력을 사용하지만 결과는 랜덤
   - 우선순위: 일반 시민팀 역할과 동일하게 설정
   - 구현 방법: 랜덤한 결과를 반환하는 특수 로직 추가

3. **폭탄 (Bomber)**
   - 밤에 사망 시 공격자도 함께 죽임
   - 우선순위: 사망 이벤트 처리 시 특수 로직 필요
   - 구현 방법: on_death 메서드 오버라이드하여 공격자도 사망 처리

4. **마녀 (Witch)**
   - 한 명에게 저주를 걸어 행동불능 상태로 만듦
   - 우선순위: 마피아 공격(50)보다 낮게 설정 (예: 40)
   - 구현 방법: 대상 플레이어에게 silenced 상태 부여

## 중립팀 역할
5. **설계자 (Architect)**
   - 직업을 예측하여 성공 시 대상을 게임에서 제거
   - 우선순위: 중간 정도 (예: 45)
   - 구현 방법: 예측 성공 시 대상 플레이어를 게임에서 제거하는 특수 로직 추가

## 마피아 서브 직업 시스템
- 마피아 클래스에 서브 직업 속성 추가
- 서브 직업으로 기자, 선동가, 시민 중 하나를 랜덤 할당
- 각 서브 직업의 능력을 마피아가 사용할 수 있도록 구현
- 마피아는 밤에 한 명을 죽이거나 서브 직업 능력을 사용할 수 있음

## 구현 계획
1. 각 역할 클래스 구현
2. 마피아 서브 직업 시스템 구현
3. 게임 메커니즘 업데이트
4. 테스트 및 디버깅
5. 문서 업데이트
